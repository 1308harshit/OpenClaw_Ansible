This directory will contain the pre-expired certificate for demo purposes.

The expired certificate will be generated by the vault_install_configure.yml playbook
using the following OpenSSL command:

openssl req -x509 -newkey rsa:2048 -nodes \
  -keyout /etc/nginx/ssl/expired_demo_cert.key \
  -out /etc/nginx/ssl/expired_demo_cert.crt \
  -days 1 \
  -subj "/C=US/ST=Demo/O=Expired/CN=timlam007.com" \
  -set_serial 01

Then the certificate's validity dates will be backdated to 2020 for demo purposes.
This file will be used by ssl_cert_make_expire.yml to simulate an expired certificate scenario.
